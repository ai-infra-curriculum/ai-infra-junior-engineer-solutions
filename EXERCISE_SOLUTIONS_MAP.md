## Solution Directory Structure Template

Each solution should follow this structure:

```
modules/mod-XXX-module-name/exercise-XX/
├── README.md                    # Solution overview and approach
├── IMPLEMENTATION_GUIDE.md      # Step-by-step implementation guide
├── src/                         # Source code
│   ├── __init__.py
│   └── solution.py
├── tests/                       # Test files
│   ├── __init__.py
│   └── test_solution.py
├── docs/                        # Additional documentation
│   └── architecture.md
├── examples/                    # Example usage
│   └── example.py
├── requirements.txt             # Python dependencies
└── Dockerfile                   # If containerization needed
```
